# JavaScript To-Do List Application

## Project Overview

This project is an interactive client-side To-Do List application developed using HTML, CSS, and JavaScript.

The application demonstrates DOM manipulation, event handling, dynamic rendering, local storage persistence, and state management.

Tasks remain saved even after refreshing or reopening the browser.

---

## Features

- Create new tasks
- View existing tasks
- Update task completion status
- Delete tasks
- Store tasks using Local Storage
- Dynamic DOM rendering
- Task filtering:
  - All
  - Active
  - Completed

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage API

---

## Functionalities

### Create
Add new tasks to the list.

### Read
Display stored tasks automatically.

### Update
Mark tasks as completed.

### Delete
Remove tasks permanently.

### Persistence
Data is stored using `window.localStorage`.
